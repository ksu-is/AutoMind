# Project Roadmap

## Phase 1 — Setup
- Add tables for leads, vehicles, and deal history

## Phase 2 — Core Features
- Lead entry form (name, contact, interest, notes)
- Vehicle inventory table (year, make, model, price, days on lot)
- Payment calculator (price, down payment, APR, term)
- Lead follow-up tracking
- Update or delete lead entries
- Sort/filter vehicles by age on lot, price, or status

## Phase 3 — UI Development
- Build homepage dashboard
- Add lead management page
- Add inventory management page
- Connect UI to database
- Create simple, clean Bootstrap styling
- Add edit/delete buttons for leads and vehicles

## Phase 4 — Optional Enhancements
- Auto-generated follow-up text suggestions
- Lead scoring based on customer messages
- Email or SMS reminders for follow-ups
- Reporting: most viewed cars, fastest-selling inventory
- Analytics charts (sales patterns, lead conversion rates)
